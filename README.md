# Algebra Learning System

Информационная система по школьному предмету «Алгебра», разработанная в рамках учебной пракики УП 01.01.

## 📖 Описание проекта

Приложение предназначено для изучения трёх тем по алгебре и содержит встроенные инструменты для решения квадратных уравнений и построения таблицы квадратов.

### Темы:
- **Понятие алгебраической дроби**
- **Упрощение рациональных выражений**
- **Понятие квадратного корня**

### Инструменты:
- **Калькулятор квадратных уравнений** (метод дискриминанта и теорема Виета)
- **Таблица квадратов чисел**

## 🛠️ Технологии

| Технология | Назначение |
|:---|:---|
| C# (.NET Framework 4.7.2) | Язык программирования |
| WPF (MVVM) | Графический интерфейс |
| PostgreSQL | База данных |
| Npgsql 7.0.10 | Драйвер для подключения к PostgreSQL |
| NUnit 3.13.3 | Модульное тестирование |
| Visual Studio 2022 | Среда разработки |

## ✨ Функционал

- 📚 Три учебные темы с форматированным содержимым
- 🏷️ Система тегов для разметки контента (`h`, `b`, `p`, `example`, `formula`)
- 🔄 Удобная навигация между темами и инструментами
- 🧮 Решение квадратных уравнений двумя методами с пошаговым выводом
- 📊 Генерация таблицы квадратов в заданном диапазоне
- ✅ Валидация вводимых данных с выводом сообщений об ошибках
- 🗄️ Хранение учебного контента в базе данных PostgreSQL
- 🧪 4 unit-теста для проверки парсера контента
- 📝 Асинхронная загрузка данных (БД → парсинг → отображение)

## 🏗️ Архитектура проекта

InventoryWpfApp/  
│  
├── ContentBlockModel.cs # Модель блока контента (тег + текст)  
├── TopicModel.cs # Модель темы  
├── DatabaseService.cs # Сервис работы с PostgreSQL  
├── ContentParser.cs # Парсер тегов в FlowDocument  
├── RelayCommand.cs # Реализация ICommand для MVVM  
├── Converters.cs # Конвертеры для WPF-биндинга  
├── MainViewModel.cs # Главная ViewModel  
├── TopicViewModel.cs # ViewModel для отображения темы  
├── MainWindow.xaml/.cs # Главное окно приложения  
├── TopicView.xaml/.cs # Представление темы  
├── EquationToolView.xaml/.cs # Калькулятор уравнений  
├── SquareTableView.xaml/.cs # Таблица квадратов  
└── App.xaml # Ресурсы приложения  

AlgebraApp.Tests/  
└── ContentParserTests.cs # Unit-тесты парсера (4 теста)
